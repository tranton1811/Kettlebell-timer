# Kettlebell Timer

A simple HTML-based kettlebell exercise timer designed for personal training routines.

## Why this project
I practice kettlebell training regularly and wanted a simple timer that matches my own workout structure.
This project helps me follow exercise sequences and track timing without relying on complex apps.

## Features
- Lightweight and easy to use
- Runs directly in the browser
- No dependencies or installation required
- Custom exercise structure based on personal kettlebell workouts

## Usage
1. Download or clone the repository
2. Open `kettlebell-timer.html` in your browser
3. Start the timer and train

## Notes
- The exercise structure is based on personal training preferences
- The project is intended for learning, sharing, and personal use
